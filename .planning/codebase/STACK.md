# Tech Stack

**Date:** 2026-05-01

## Core Technologies
- **Frontend Framework:** React 18 (via `react` and `react-dom`)
- **Build Tool:** Vite (via `vite` and `@vitejs/plugin-react`)
- **Language:** JavaScript (ES modules, `.jsx` for components)
- **Styling:** Plain CSS with variables (`index.css`)

## Dependencies
- `@supabase/supabase-js`: Backend-as-a-Service integration for auth/database.

## Configuration
- `vite.config.js`: Vite build configuration
- `package.json`: Project metadata and dependency management (`"type": "module"`)
- `jsconfig.json`: IDE JS configuration
